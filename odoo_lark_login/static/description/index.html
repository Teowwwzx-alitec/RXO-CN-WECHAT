Lark login （飞书登录）

This module provides Odoo and Lark docking.（该模块提供了Odoo和飞书对接。）

17版本进行了优化，增加了请求转发（可以实现一个应用多个域名使用），飞书绑定。

Usage（使用）

To use this module, you need to:（要使用此模块，您需要:）

安装python依赖：
"simplejson"
安装飞书模块：
在设置菜单中增加飞书配置参数，值得注意的是return_url和bind_url是授权回调域+固定地址。 目的是请求转发，实现一个应用可以多个域名使用。
替换客户ID为appid。
飞书绑定：在首选项中增加飞书绑定按钮，为当前用户绑定个人飞书。
扫码登录：登录页面增加了飞书登录
点击飞书登录，自动跳转至飞书扫码页面，扫码确认后自动登录。
